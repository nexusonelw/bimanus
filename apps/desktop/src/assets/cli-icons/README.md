# CLI brand icons (local assets)

Static monochrome SVG marks used by `src/cli-icons.tsx` (compiled into the renderer; no runtime download).

| File | CLI | Source |
| --- | --- | --- |
| `codex.svg` | CodeX | Lobe Icons (`@lobehub/icons-static-svg`) Codex mark |
| `claude.svg` | Claude Code | Lobe Icons Claude Code mark |
| `opencode.svg` | OpenCode | Lobe Icons opencode mark |
| `grok.svg` | Grok | Lobe Icons Grok mark |
| `copilot.svg` | Copilot | Lobe Icons GitHub Copilot mark |
| `antigravity.svg` | Antigravity | Lobe Icons Antigravity mark |
| `kiro.svg` | Kiro | Lobe Icons Kiro mark |
| `cursor.svg` | Cursor | Lobe Icons Cursor mark |
| `droid.svg` | Droid (Factory) | Factory.ai favicon SVG monogram |

All icons use `fill="currentColor"` so UI theme / brand accent color can be applied via CSS `color`.
